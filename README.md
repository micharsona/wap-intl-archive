# wap-intl-archive

## 项目简介

`wap-intl-archive` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
这些页面原本来源于移动端或国际化场景下的特定功能界面，经过整理后以静态 HTML 形式保存于此。  
本仓库不针对任何单个域名或具体网站，仅作为页面集合的存档与展示用途。

## 目录说明

```
.
├── pages/          # 存放归档的 HTML 页面文件
├── assets/         # 页面引用的静态资源（样式、脚本、图片等）
├── index.html      # 归档页面索引（可选，用于快速浏览）
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件对应一个独立的归档页面。文件名或目录名通常反映页面的原始功能或版本号。
- `assets/`：如果页面依赖外部资源，会尽量内联或本地化存放于此目录，以保证离线可访问性。

## 页面归档说明

归档的 HTML 页面满足以下特征：

- 每个页面均为独立完整的 HTML 文件，包含必要的样式与脚本（或引用本地资源）。
- 页面内容保持原样，仅作必要的路径修正（如将外部链接转为本地引用）。
- 不包含任何动态后端依赖，可直接在浏览器中打开运行。

### 如何查看归档页面

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/wap-intl-archive.git
   ```
2. 进入 `pages/` 目录，直接双击任意 HTML 文件即可在浏览器中打开查看。
3. 如果仓库根目录包含 `index.html`，也可通过本地 Web 服务器（如 `python -m http.server`）启动后浏览。
